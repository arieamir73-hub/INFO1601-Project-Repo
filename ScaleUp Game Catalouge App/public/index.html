Fequently asked Questions

What is Skal Reviews V

Skal Reviews is a free game discovery platform that helps you find and learn about new, popular, and genre-specific games before you play them.

Where does the game data come from V

We pull game information from the FreeToGame API and Steam API to provide accurate details, screenshots, and release information.

What are the benefits of creating an account V

With an account, you can favorite games, save them to your profile, and quickly access your curated game list anytime you return.

Why isn't a specific game showing up V

We primarily feature free-to-play games. If a paid game isn't appearing, it may not be available in our current data sources.

How do I search for a game V

Use the search bar at the top of the page. Type a game name and press Enter to find matching results

Can I suggest a game to be added V

Currently we don't accept manual submissions, but we're always expanding our database through API updates.
